## Utility Matrix Buiilder

### Build

The program can be built as follows:
```
mvn package
```

This generates `target/utility-matrix-builder-1.0-SNAPSHOT.jar`.

### Execute

The artifact `target/utility-matrix-builder-1.0-SNAPSHOT.jar` can be executed as follows:
```
java -cp target/utility-matrix-builder-1.0-SNAPSHOT.jar es.ull.gco.UtilityMatrixBuilder 15 65
```
